# ppqq-on-desk v0.10.1-ppqq.9

This prerelease bundles the PP and QQ desktop pet themes and makes PP the default for fresh installs.

Changes:

- Add the `pp` theme as a built-in bundled theme.
- Add the legacy `qq` theme as a built-in bundled theme.
- Change the fresh-install default theme from `clawd` to `pp`.
- Keep both bundled themes validated against the 192x208 animated WebP runtime format.
