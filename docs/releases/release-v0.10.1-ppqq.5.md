# ppqq-on-desk v0.10.1-ppqq.5

This prerelease bundles the Q Pet character theme directly with ppqq-on-desk.

Changes:

- Add `q-pet` as a built-in desktop pet theme.
- Include the finalized 192x208 Q Pet animated WebP runtime assets.
- Keep Q Pet available immediately after installing the app, without a manual theme copy step.
- Preserve the ppqq-on-desk release, update, and hook metadata from the previous prereleases.
