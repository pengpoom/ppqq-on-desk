# ppqq-on-desk v0.10.1-ppqq.10

This prerelease cleans up the built-in QQ desktop pet theme package for fresh installs.

Changes:

- Remove the old built-in `q-pet` theme folder so fresh installs only include the canonical `qq` theme.
- Rename the `qq` theme display name from `Q Pet` to `QQ`.
- Align the `qq` theme idle/sleep cadence with `pp`: 2-minute random idle actions and 5-minute sleep trigger.
- Expand the `qq` idle random pool to use the available belly-pat, daydream, and tidy animations.
